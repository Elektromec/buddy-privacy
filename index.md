## 2. Welche Daten werden erhoben?

Die Buddy App erhebt folgende Daten:

- **Benutzername**: Der von Ihnen gewählte Anzeigename
- **Standortdaten**: Ihr Standort wird mit Gruppenmitgliedern geteilt (nur wenn Sie einer Gruppe beitreten)
- **Chat-Nachrichten**: Nachrichten innerhalb Ihrer Gruppen
- **Push-Token**: Für Benachrichtigungen

## 3. Zweck der Datenverarbeitung

- Anzeige Ihres Standorts auf der Gruppenkarte
- Zustellung von Chat-Nachrichten
- Senden von Push-Benachrichtigungen

## 4. Datenweitergabe

Ihre Daten werden nur mit Mitgliedern Ihrer Gruppe geteilt. Es erfolgt keine Weitergabe an Dritte.

## 5. Datenspeicherung

- Daten werden auf sicheren Cloud-Servern gespeichert
- Standortdaten werden nur während der aktiven Gruppenmitgliedschaft gespeichert
- Beim Verlassen einer Gruppe werden Ihre Standortdaten gelöscht

## 6. Ihre Rechte

Sie können jederzeit:
- Die App deinstallieren
- Eine Gruppe verlassen
- Die Löschung Ihrer Daten anfordern
